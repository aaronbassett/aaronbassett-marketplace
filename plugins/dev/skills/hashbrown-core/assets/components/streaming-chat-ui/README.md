# Streaming Chat UI Component

This component provides a simple chat interface that can render a streaming response from a compatible server, like the `streaming-chat-server` template.

## Usage

1.  **Integrate the Component:**
    Copy the `StreamingChatUI.tsx` file into your React project.

2.  **Connect to a Server:**
    Make sure the `fetch` request in the component points to your streaming server endpoint.

## Dependencies

This component uses the `@hashbrown/react` library for state management.

```bash
npm install @hashbrown/react
```
