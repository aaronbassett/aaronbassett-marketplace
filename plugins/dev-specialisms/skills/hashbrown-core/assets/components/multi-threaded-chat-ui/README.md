# Multi-Threaded Chat UI Component

This component provides a UI for managing multiple chat conversations (threads), complementing a server like the `chat-server-with-threads` template.

## Usage

1.  **Integrate the Component:**
    Copy the `MultiThreadedChatUI.tsx` and the `hooks` directory into your React project.

2.  **Connect to a Server:**
    Update the API endpoints in the `useThreads.ts` hook to point to your server.

## Dependencies

This component uses the `@hashbrown/react` library for state management.

```bash
npm install @hashbrown/react
```
