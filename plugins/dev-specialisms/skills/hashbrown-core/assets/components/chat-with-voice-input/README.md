# Chat with Voice Input Component

This component extends a standard chat interface with a button to accept voice input using the browser's Web Speech API.

## Usage

1.  **Integrate the Component:**
    Copy the `ChatWithVoiceInput.tsx` file into your React project.

2.  **Browser Compatibility:**
    The Web Speech API is not supported in all browsers. Check browser compatibility before use.

## Dependencies

This component uses the `@hashbrown/react` library for state management.

```bash
npm install @hashbrown/react
```
