# Ecosystem Tools

Essential packages for React development.

## Routing

**React Router** - Industry standard
```bash
pnpm add react-router-dom
```

**TanStack Router** - Type-safe alternative
```bash
pnpm add @tanstack/react-router
```

## Data Fetching

**React Query** - Server state management
```bash
pnpm add @tanstack/react-query
```

**SWR** - Lightweight alternative
```bash
pnpm add swr
```

## Forms

**React Hook Form** - Performant, flexible
```bash
pnpm add react-hook-form
pnpm add @hookform/resolvers zod
```

## State Management

**Zustand** - Lightweight, simple
```bash
pnpm add zustand
```

**Jotai** - Atomic state
```bash
pnpm add jotai
```

## Styling

**Tailwind CSS** - Utility-first
```bash
pnpm add -D tailwindcss postcss autoprefixer
```

**CSS Modules** - Scoped CSS (built into Vite)

## UI Components

**Radix UI** - Unstyled, accessible
```bash
pnpm add @radix-ui/react-dialog @radix-ui/react-dropdown-menu
```

**shadcn/ui** - Copy-paste components
```bash
npx shadcn@latest init
```

## Icons

**Lucide React** - Modern icon library
```bash
pnpm add lucide-react
```

## Date Handling

**date-fns** - Modern, tree-shakeable
```bash
pnpm add date-fns
```

## Utilities

**clsx** - Conditional classnames
```bash
pnpm add clsx
```

**es-toolkit** - Modern lodash alternative
```bash
pnpm add es-toolkit
```

## Development

**TypeScript** - Type safety
```bash
pnpm add -D typescript @types/react @types/react-dom
```

**ESLint** - Linting
```bash
pnpm add -D eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin
pnpm add -D eslint-plugin-react eslint-plugin-react-hooks
```

**Prettier** - Formatting
```bash
pnpm add -D prettier
```

## Choose packages based on needs, not hype. Start minimal, add as required.
